# 📊 Financial Sales Analytics Dashboard

An end-to-end **Financial Sales Analytics** project developed using **Python, MySQL, and Power BI**. This project demonstrates the complete data analytics workflow, from data cleaning and analysis to database management and interactive dashboard creation.

---

## 🚀 Project Overview

The objective of this project is to analyze financial sales data and create an interactive dashboard that helps visualize business performance, sales trends, profitability, and regional insights.

---

## 🎯 Objectives

- Clean and preprocess raw financial data using Python.
- Store and manage the dataset using MySQL.
- Analyze sales and profit performance.
- Build an interactive Power BI dashboard.
- Generate business insights through data visualization.
